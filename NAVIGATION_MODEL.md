- Reference Data

- Global Templates
- Website Templates
- Feature Configuration
- Knowledge Review Queue

## Agency Staff Navigation

- Dashboard
- Clients
- Passengers
- Requests
- Offers
- Bookings
- Tickets
- EMDs
- Invoices
- Payments
- Refunds / Exchanges
- Documents
- Messages
- Tasks
- Airline Intelligence
- Agency Knowledge / Overrides
- Website
- Templates
- Branding
- Staff
- Settings

Recommended agency dashboard signals:

- Open requests.
- Offers awaiting client response.
- Upcoming departures.
- Unconfirmed services.
- Documents needed.
- Invoices due or overdue.
- Payment reminders.
- Refund/exchange cases in progress.

## Client Portal Navigation

- Dashboard
- Requests
- Offers
- Trips / Bookings
- Itinerary Timeline
- Passengers
- Documents
- Tickets
- EMD Receipts
- Invoices / Payments
- Refunds / Exchanges
- Messages
- Tasks
- Profile

Client portal must be responsive and mobile-friendly from the first version.

## Public Agency Website Navigation

- Home
- Services
- About
- Guides / Blog optional
- FAQ
- Contact
- Request Travel
- Client Login
- Register optional
- Legal pages

Website navigation should stay controlled and template-driven. The goal is a fast, professional agency presence, not a complex custom site builder.
